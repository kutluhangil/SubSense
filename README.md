## ✨ Transform the Way You Manage Subscriptions

Subscriptions creeping up on you? **SubSense** puts you in complete control. It's more than just an expense tracker—it computes currency conversions, monitors pricing across different regions, and uses Google Gemini AI to analyze your spending habits and offer optimization strategies—all in a premium, beautifully designed interface.

## 🛠 Tech Stack

SubSense is built using modern, industry-standard technologies to ensure extreme performance and reliability.

- **Frontend:** React 19, TypeScript, Vite, TailwindCSS, Framer Motion
- **Backend & Auth:** Firebase Auth, Firestore, Firebase Cloud Functions
- **AI Integration:** Google GenAI (Gemini)
- **Payments:** Stripe
- **Icons:** Lucide React, Google Material Icons

## 🏁 Getting Started

Want to run SubSense locally? Follow these steps to get your local environment up and running.

### Prerequisites
- Node.js (v18 or higher recommended)
- Firebase Project (for Authentication & Database)
- Gemini API Key (for AI Insights)

### Installation

<ol>
  <li>
    <strong>Clone the repository</strong>
    <br/>
    <pre><code>git clone https://github.com/your-username/SubSense.git<br/>cd SubSense</code></pre>
  </li>
  <li>
    <strong>Install dependencies</strong>
    <br/>
    <pre><code>npm install</code></pre>
  </li>
  <li>
    <strong>Configure Environment Variables</strong><br/>
    Create a <code>.env.local</code> file in the root directory and add your API keys:
    <br/>
    <pre><code>VITE_GEMINI_API_KEY=your_gemini_api_key_here<br/># Add Firebase & Stripe config vars as needed</code></pre>
  </li>
  <li>
    <strong>Start the development server</strong>
    <br/>
    <pre><code>npm run dev</code></pre>
    <p>Now open <a href="http://localhost:5173">http://localhost:5173</a> to view it in your browser.</p>
  </li>
</ol>
